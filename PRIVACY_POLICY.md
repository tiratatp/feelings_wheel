# Privacy Policy

**Feelings Wheel** (com.nuttyknot.feelingswheel)

*Last updated: March 29, 2026*

## Overview

Feelings Wheel is a fully offline Android application. It does not collect, store, or transmit any data whatsoever.

## Data Collection

This app does **not** collect any personal data or usage data. Specifically:

- **No personal information** is collected (name, email, location, etc.)
- **No usage analytics** or tracking of any kind
- **No crash reporting** or diagnostics
- **No cookies** or similar technologies

## Network Access

This app does **not** access the internet. It requests no network permissions and makes no network calls. There are no servers, APIs, or cloud services involved.

## Third-Party Services

This app uses **no** third-party services, SDKs, or libraries that collect data.

## Data Storage

All data within the app is **ephemeral** — it exists only in memory while the app is running and is lost when the app is closed. There is no persistent storage, no local database, and no files written to your device.

## Permissions

This app requests **no special permissions** beyond the standard Android defaults required to display on screen.

## Children's Privacy

Because this app collects no data of any kind, it is safe for users of all ages.

## Changes to This Policy

If this policy is updated, the changes will be reflected in this file with an updated date.

## Contact

If you have questions about this privacy policy, you can reach the developer through the project's GitHub repository.
